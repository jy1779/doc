# git 指令

> git clone 
>
> 指定目录,如果指定目录不存在，则自动创建，如果目录存在会报错

```shell
git clone ssh://git@zeno-gitlab.xw18.cn:2222/zeno/zeno-config-yaml.git "指定目录"
```

> git add
>
> 指定目录添加

```shell
git add /tmp/test/
```

> git commit
>
> git 提交

```shell
git commit -m "add test"
```

> git push
>
> git 上传

```shell
git push -u origin master
```
